## Account Setup

### Creating Your Account

1. **Visit the homepage** at your OxFin domain
2. **Click "Sign Up"** in the top right corner
3. **Enter your details**:
   - Full name
   - Valid email address
   - Strong password (minimum 12 characters)
4. **Verify your email** through the confirmation link sent to your inbox
5. **Complete profile setup** by adding:
   - Phone number (optional)
   - Profile picture (optional)

### Login Options

**Email & Password:**
- Use your registered email and password
- Account lockout after 5 failed attempts

**Multi-Factor Authentication (Recommended):**
- Enable 2FA in Settings > Security
- Use Google Authenticator or similar app
- Scan the QR code provided
- Enter the 6-digit code to verify

**Social Login:**
- Google Sign-In
- GitHub Authentication

## Wallet Management

### Wallet Types

**Full Wallet:**
- Higher daily transaction limits
- Interest earning capabilities
- Advanced features unlocked
- Suitable for active users

**Lite Wallet:**
- Basic transaction capabilities
- Lower fees
- Simplified interface
- Perfect for beginners

### Creating Wallets

1. Navigate to **Wallets** in the sidebar
2. Click **Create New Wallet**
3. Choose wallet type (Full/Lite)
4. Select currency (INR/OX)
5. Set initial deposit amount (optional)

### Wallet Features

**Balance Management:**
- Real-time balance updates
- Currency conversion between INR and OX
- Transaction history with filtering
- Export statements in CSV/PDF

**Transfer Funds:**
1. Click **New Transfer** on dashboard or Wallets page
2. Select source wallet
3. Choose destination wallet
4. Enter amount and description
5. Confirm with security verification

**Transaction History:**
- Filter by date range
- Search by description
- Categorize transactions
- Export to spreadsheet
